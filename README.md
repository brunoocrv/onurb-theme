# önurb theme

A minimal theme inspired in Min Theme and Dracula.

## Installation

1. Install theme from the [Marketplace](https://marketplace.visualstudio.com/items?itemName=brunoocrv.onurb-theme)
2. Go to `File > Preferences > Color Theme`
3. Select `önurb dark`
